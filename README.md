# Python Learning Basics

A structured collection of Python fundamentals practice files covering core programming concepts, syntax, and beginner-level problem solving.

## Overview

This repository contains my early Python practice work, organized by topic. The goal was to build a strong foundation in Python through hands-on exercises, small examples, and simple programs.

## Topics Covered

- Variables and strings
- Numbers and basic calculations
- Lists and tuples
- Dictionaries
- Conditional statements
- Loops
- Functions
- File handling
- Error management
- Classes and objects
- Basic testing
- Simple quiz-style programs
